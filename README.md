# 🛠 План розробки проєкту: 1% краще кожен день"
 
## 1. Концепція продукту
- **Тип додатку:** Чат-бот
- **Основна ідея:** бот щодня дає користувачу маленьке завдання для саморозвитку: спорт, мислення, звички або продуктивність. Також бот відстежує прогрес і мотивує користувача ставати на 1% кращим щодня.
 
## 2. Основний функціонал (Features)
- [ ] Приймати текст від користувача
- [ ] Робити математичний розрахунок
- [ ] Виводити результат
 
## 3. Структура даних
| Назва змінної      | Тип даних | Для чого потрібна             |
| :----------------- | :-------- | :---------------------------- |
| user_input       | str     | Ввід тексту від користувача   |
| is_active        | bool    | Контроль роботи бота          |
| task_category    | str     | Категорія завдання            |
| daily_task       | str     | Завдання на день              |
| completed_tasks  | int     | Кількість виконаних завдань   |
| progress_percent | float   | Відсоток прогресу користувача |
| user_name        | str     | Ім’я користувача              |
